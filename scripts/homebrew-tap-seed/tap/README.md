# homebrew-oat

Official [Homebrew](https://brew.sh) tap for [**OAT** (Open Agent
Teams)](https://github.com/Root-IO-Labs/open-agent-teams) — orchestrate
parallel coding agents on GitHub repos.

## Install

```bash
brew tap Root-IO-Labs/oat
brew install oat
```

This installs both the `oat` CLI and the `oat-agent` helper binary.

## Upgrade

```bash
brew update
brew upgrade oat
```

## Contributing

**Please do not open PRs against this repo.** The formula here is
generated by [GoReleaser](https://goreleaser.com) on each tagged
release of
[`Root-IO-Labs/open-agent-teams`](https://github.com/Root-IO-Labs/open-agent-teams),
and hand-edits will be overwritten on the next release.

If you spot a problem with the formula, file an issue in the main repo:

<https://github.com/Root-IO-Labs/open-agent-teams/issues>
