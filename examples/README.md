# Examples

Short, self-contained walkthroughs you can run against a fresh OAT install.
Start with [first-run.md](first-run.md) if you've never used OAT before.

| Example | What it shows |
|---|---|
| [first-run.md](first-run.md) | Clone a toy repo, spin up a supervisor + merge-queue, watch one worker open and merge a PR. End-to-end in ~10 minutes. |

More examples will land as the project matures. If you have an idea for one,
open an issue — concrete "what should this do?" examples are always welcome.
